# Northwind Traders Sales & Performance Dashboard

## Overview

This project was developed during my Data Analytics Training at ICT Academy - Rockwell Automation.

The dashboard was created in Power BI using the Northwind Traders dataset to analyze sales performance, customer behavior, product performance, employee productivity, and shipping operations.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel

## Key Metrics

- Gross Revenue: 1.35M
- Net Revenue: 1.27M
- Total Orders: 830
- Total Quantity: 51K
- Average Shipping Days: 8.49
- YoY Revenue Growth: 179.31%

## Dashboard Features

- Revenue Analysis
- Customer Analysis
- Product Analysis
- Employee Performance Tracking
- Shipping Delay Analysis
- Country-wise Revenue Analysis
- Quarterly and Monthly Revenue Trends

## Business Insights

- USA generated the highest revenue.
- Beverages was the top-selling category.
- Q1 2015 was the best-performing quarter.
- Shipment delay rate was only 4.46%.
- Top customers contributed significantly to overall revenue.
